pipeline.py is the primary script used for analyzing the recorded data.  
main_functions.py and plotting_functions.py contain functions that required to execute pipeline.py
The main recording data is stored in the example_data folder. (see Display HDF5 data file)
In the Display HDF5 data file, please focus on  CMN_no_foreground (it is CMN motion vectors creating CMN visual stimulus on the sphere). And phase0 is the phase  when CMN Stimulus is shown on the sphere. Other data can be disregarded. 
Before running the pipeline, set a designated save path and copy its address into pipeline.py. Create a folder named 'png' and a folder named 'pdf' in the save path.
